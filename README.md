# Evolución de la Ganadería Vacuna Argentina(1980-2025)
Descripción 
Este proyecto tiene como objetivo analizar la evolución de la ganadeía vacuna argentina durante las últimas décadas utilizando datos oficiales provenientes de organismospúblicos nacionales. 

Se desarrollará un flujo de trabajo completo de anális de datos, incluyendo:
  * Investigación y recopilación de datos.
  * Diseño de una base de datos relacional.
  * Integración y limpieza de datos.
  * Consultas SQL.
  * Análisis Exploratorio de Datos(EDA).
  * Visualización de resultados.
  * Elaboración de conclusiones basadas en evidencia.

Objetivos
  * Analizar la evolución del stock bovino.
  * Estudiar la faena y la producción de carne.
  * Analizar el consumo interno.
  * Evaluar la evolución de las exportaciones.
  * Identificar tendencias y cambios relevantes en el sector.

Tecnologías 
  * PostgreSQL
  * SQL
  * Python
  * Pandas
  * NumPy
  * Matplotlib
  * Jupyter Notebook
  * Git
  * GitHub
